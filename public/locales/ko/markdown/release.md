## 🚀 업데이트 예정

더 나은 분석 환경을 제공하기 위해 준비 중인 기능들입니다.

최대한 신속하고 안정적으로 업데이트하겠습니다 :)

- **분석 세션 관리**
  - 분석 결과 저장/내보내기/불러오기
- **사용자 맞춤 환경설정**
  - 사용자별 세부적인 작업 환경 설정
- **검색 기능 고도화**
  - 더욱 빠르고 정밀한 데이터 탐색을 위한 검색 로직 개선
- **분석 포맷 확장**
  - 기존 이미지 파일 외에도 다양한 형태의 파일 분석 지원
- **데이터 편집기**
  - 헥스(Hex) 데이터 및 메타데이터 직접 수정 및 편집 기능
- **인증 및 확장 도구**
  - 사용자 인증 시스템 도입 및 심층 분석을 위한 추가 도구 제공
