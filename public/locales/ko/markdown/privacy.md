# 개인정보 처리방침

**최종 업데이트: 2026년 3월 03일**

---

## 1. 개요

**ICE Forensic**은 웹 기반 헥스 뷰어 및 EXIF 분석 도구를 제공하며, 이용자의 프라이버시와 데이터 보안을 최우선으로 보호합니다. 본 개인정보 처리방침은 서비스 이용 시 수집되는 정보의 종류, 용도, 보호 방식 등을 설명합니다.

## 2. 수집하는 정보 및 수집 방법

서비스의 원활한 제공과 보안 유지를 위해 아래와 같은 최소한의 정보가 서비스 이용 과정에서 자동으로 생성되어 수집될 수 있습니다.

- **자동 수집 정보:** 브라우저 종류 및 버전, 운영체제(OS), 방문 일시 등 기기 및 환경 정보 _(※ IP 주소 등은 당사가 직접 수집하지 않으며, 타사 호스팅 및 분석 도구의 정책에 따라 처리됩니다.)_
- **통계 및 오류 데이터:** 웹사이트 트래픽, 사용성 분석 및 **서비스 오류 진단(예: 에러 코드, 토스트 메시지 식별자)을** 위한 타사 분석 도구(Google Analytics) 데이터
- **사용자 요청 시 제3자 제공(전송) 데이터:**
  - **IP 주소:** 사용자가 '내 IP 조회' 기능을 실행(버튼 클릭)할 경우, 해당 정보 조회를 위해 사용자의 실시간 IP 주소가 외부 API 서비스인 **ipinfo.io**로 전송됩니다.
  - **위치 정보(위도·경도):** 사용자가 사진의 위치 정보 조회를 요청할 경우, 지도 시각화 및 주소 변환(Reverse Geocoding)을 위해 해당 좌표 데이터가 **Nominatim (OpenStreetMap)** 및 **Leaflet** 연동 서버로 전송됩니다.

**※ 수집 및 전송하지 않는 정보:** 사용자가 분석을 위해 브라우저에 불러오는 **파일의 원본 내용, 파일명, (조회를 요청하지 않은) 메타데이터 및 사용자의 이름, 이메일 등 개인을 식별할 수 있는 정보는 당사가 일절 수집하거나 당사 서버로 전송하지 않습니다.**

## 3. 개인정보의 처리 목적

수집된 정보는 다음의 목적을 위해서만 활용됩니다.

- 서비스의 원활한 운영, 안정성 유지 및 사용자 경험(UX) 개선
- 비정상적인 접근 차단, 보안 위협 대응 및 통계적 분석
- 관련 법령에 따른 법적 의무 이행 및 분쟁 해결

## 4. 데이터 처리 및 보안 방식 (중요)

본 서비스는 자체 웹 서버나 데이터베이스를 통해 이용자의 민감한 데이터를 수집·보관하지 않으며, 아래의 원칙에 따라 데이터를 안전하게 처리합니다.

- **100% 브라우저 로컬 처리:** 파일 데이터(Hex, EXIF 등) 분석은 WebAssembly(Wasm) 및 브라우저 API를 활용하여 이용자의 기기(브라우저) 내에서만 단독으로 실행되며, 어떠한 경우에도 원본 파일이 외부로 전송되거나 공유되지 않습니다.
- **명시적 동의에 의한 외부 API 호출:** IP 조회 및 위치(주소) 변환 기능은 **이용자가 명시적으로 해당 기능을 요청(버튼 클릭 등)한 경우에 한하여 작동**합니다. 백그라운드에서 임의로 사용자의 위치나 IP 정보를 외부로 전송하지 않습니다.
- **제3자 서비스의 보호 정책:** 기능 수행을 위해 외부(ipinfo.io, OpenStreetMap 등)로 전송된 한시적 데이터(IP, 좌표 등)는 각 해당 서비스 제공자의 개인정보 처리방침을 따르며, 당사는 해당 데이터를 별도로 저장, 가공, 또는 타 데이터와 결합하지 않습니다.
- **익명화된 통계 수집:** 서비스 개선을 위해 Google Analytics로 전송되는 이벤트 및 오류 로그(`message_code` 등)는 개인을 식별할 수 없도록 철저히 익명화되어 전송됩니다.

## 5. 개인정보의 보유 및 파기

ICE Forensic은 GitHub Pages를 통해 정적(Static) 웹사이트 형태로 제공되며, 자체적인 웹 서버나 데이터베이스를 운영하지 않습니다. 따라서 통신비밀보호법 등에서 규정하는 **이용자의 웹사이트 접속 로그(IP 주소 등)를 당사가 직접 수집하거나 보관하지 않습니다.** _(단, 웹사이트 호스팅 및 보안 유지를 위해 호스팅 제공업체인 GitHub 측에서 표준적인 시스템 로그를 수집할 수 있으며, 이는 GitHub의 개인정보 처리방침을 따릅니다.)_

서비스 품질 향상 및 오류 진단을 위해 타사 분석 도구(Google Analytics)를 통해 수집되는 익명화된 통계 및 에러 데이터(`message_code` 등)는 **최대 14개월간** 보관되며, 동일한 브라우저에서 새로운 활동 발생 시 보관 기간이 갱신됩니다. 보관 기간이 경과한 데이터는 해당 서비스의 정책에 따라 복구할 수 없는 방법으로 영구 파기됩니다.

## 6. 쿠키(Cookies) 및 로컬스토리지(Local Storage) 사용

ICE Forensic은 이용자에게 맞춤형 환경을 제공하고 서비스 품질을 향상하기 위해 쿠키와 로컬스토리지를 사용합니다.

- **로컬스토리지 (Local Storage):** 이용자의 브라우저 환경 설정(예: 다크 모드, UI 레이아웃 등)을 유지하기 위해 사용됩니다. 이 데이터는 이용자의 기기에만 저장되며 서버로 전송되지 않습니다.
- **쿠키 (Cookies):** 웹사이트 방문자 통계 분석, 특정 기능의 오류 발생 빈도 측정, 및 비정상적인 접근 방지를 위해 타사 분석 도구(Google Analytics)에서 사용됩니다.
- **거부 방법:** 이용자는 웹 브라우저의 옵션을 설정함으로써 쿠키 저장을 거부하거나 로컬스토리지 데이터를 직접 삭제할 수 있습니다. 단, 이를 거부할 경우 설정된 환경이 초기화되거나 일부 편의 기능 사용에 제한이 있을 수 있습니다.
